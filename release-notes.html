Release Notes

Versão Atual
Lançamento
Histórico

Bug Fixes:
  • Correção de erro calculava incorretamente o ano dos dados do EPM;
  • Correção de erro que não abria páginas de Calcular EPM e Simular Meu RH em novo guia;

What's New:
  • Botões de 'Simular Meu RH' e 'Calcular EPM' reposicionados. Os botões agora também abrem em uma nova aba;
  • Melhorada gestão de versões.

Bug Fixes:
  • Correção de erro que exibia incorretamente o cargo do usuário;
  • Correção de erro que travava a criação de novos conversores EPM;
  • Correção bug que não identificava corretamente que o arquivo foi atualizado.

What's New:
  • Adicionado suporte para arquivos HTML do Meu RH;
  • Vídeos de "Como Exportar" melhorados;
  • Adicionada opção de arrastar arquivos;
  • Melhoria no tratamento de erros na importação de arquvios;
  • Adicionada opção de editar o perfil;
  • Adicionado indicador caso o usuário atualize um arquivo;
  • Adicionados botões de "Simular" e "Converter EPM" em cada dia;
  • Adicionado dado de modelo de trabalho no "Visualizar EPM";
  • Adicionada opção de adicionar múltiplos cálculos de EPM;
  • Adicionada opção de incluir foto de perfil personalizada.

Bug Fixes:
  • Correção de bug que não removia a tela de "Datas não batem" mesmo se o usuário apagasse um dos arquivos;
  • Correção problema de recursividade ao inserir vários arquivos;
  • Correção bug que o sistema identificava incorretamente incompatibilidade de versões.

What's New:
  • Adicionado link para simular caso o número de batidas seja ímpar

Bug Fixes:
  • Correção de bug que avisava que o usuário bateu ponto muito tarde caso estivesse com marcações ímpares;
  • Correção de bug que impedia usuário de simular com saldo negativo ou caso não inserisse nada no campo;
  • Correção de bug que que não identificava corretamente um dia ausente e inseria uma marcação ímpar.

What's New:
  • Todos os pontos válidos agora exibirão uma mensagem de ponto válido;
  • Novo Checkbox para o usuário ignorar, caso esteja comparando EPM e Meu RH, datas fora do período de cobertura;
  • Criada mensagem de "Não existem dados" caso o usuário aplique filtros que não retornem resultados.

Bug Fixes:
  • Correção no carregamento de páginas;
  • Correção de bug que exibia comparação EPM-MeuRH mesmo com períodos diferentes;
  • Correção de bug que ocultava o dia de hoje caso o usuário selecionasse para ignorar dias não trabalhados

Bem vindos(as) ao Ponto! 😃

Essa versão inicial busca implementar todas as funcionalidades esperadas para o sistema, porém é possível que alguns bugs ainda estejam presentes.

Caso detecte alguma falha, não exite em entrar em contato via o botão Entre em Contato no menu lateral.
